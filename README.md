# Oxford Academic Complex (OAC) Website

A modern, responsive website for Oxford Academic Complex built with React, Vite, and Tailwind CSS.

## 🚀 Features

- **Responsive Design** - Mobile-first approach
- **Interactive Navigation** - Smooth scrolling between sections
- **Modern Animations** - Framer Motion for smooth transitions
- **Professional UI** - Clean, purple-themed design
- **Contact Form** - Interactive contact section
- **Image Fallbacks** - Graceful error handling

## 🛠️ Tech Stack

- **React 18** - Modern React with hooks
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **Lucide React** - Icon library

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ (Current: v22.17.1)
- npm 8+ (Current: v10.9.2)

### Installation
1. Clone or download the project
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
```bash
npm run dev
```
The website will open at `http://localhost:3000`

### Build
```bash
npm run build
```

### Preview Build
```bash
npm run preview
```

## 📱 Sections

1. **Hero** - Main landing with background image
2. **About** - Institution overview and mission
3. **Programs** - Nursery & Primary education offerings
4. **Campuses** - Loum and Tombel locations
5. **Why Choose Us** - Benefits and advantages
6. **Contact** - Contact form and information
7. **Footer** - Quick links and company info

## 🎨 Customization

The component accepts these props:
- `animationDuration` - Animation speed (default: 0.5)
- `primaryColor` - Primary color (default: #6b21a8)
- `secondaryColor` - Secondary color (default: #4b5563)
